Overview

VARIANT is a comprehensive virus mutation analysis tool designed to track mutations at both genome and protein levels. It provides detailed analysis of viral sequences, identifying mutations, hotspots, and potential frameshifting sites.

Key Capabilities

  • Multi-sequence alignment analysis
  • Mutation detection and classification
  • Hot and row mutation identification
  • Frameshift site detection
  • Protein-level impact analysis

Supported Viruses

  • Single-segment viruses (SARS-CoV-2, HIV-1, etc.)
  • Multi-segment viruses (Influenza H3N2, etc.)
  • Custom virus definitions

Features

Genome Analysis

Comprehensive mutation tracking across entire viral genomes

Protein Impact

Analysis of mutations on protein structure and function

Frameshift Detection

Identification of potential ribosomal frameshifting sites

Quick Start

🚀 Get Started in 4 Steps

Follow these simple steps to analyze your virus data

Step 1: Create Custom Virus

  1. 1Go to Data Upload tab
  2. 2Enter a unique virus name
  3. 3Select virus type (single/multi-segment)
  4. 4Click "Create Custom Virus"

Step 2: Upload Data Files

  1. 1Select your virus from dropdown
  2. 2Upload reference genome (.fasta)
  3. 3Upload reference proteome (.fasta)
  4. 4Upload MSA file (.txt)

Step 3: Start Analysis

  1. 1Go to Analysis tab
  2. 2Select your virus
  3. 3Choose analysis options
  4. 4Click "Start Analysis"

Step 4: Download Results

  1. 1Wait for analysis completion
  2. 2Go to Files tab
  3. 3Browse result files
  4. 4Download individual files or ZIP

File Requirements

Reference Proteome
Format .fasta
Content Protein sequences
  • Multiple proteins allowed
  • Standard FASTA format
  • All expected proteins
MSA File
Format .txt
Content Multiple sequence alignment
  • ClustalW format preferred
  • Contains all sample sequences
  • Aligned sequences

Usage Guide

Analysis Options

Process All Genomes

Analyzes all sequences in the MSA file. This is the recommended option for comprehensive analysis.

  • Generates files for each genome
  • Identifies common mutations
  • Provides complete analysis
Specific Genome ID

Analyzes only one specific sequence (e.g., EPI_ISL_123456).

  • Faster processing
  • Targeted analysis
  • Single genome focus

Advanced Features

Frameshift Detection

Detects potential programmed ribosomal frameshifting sites in the genome. This feature:

  • Identifies slippery sequences
  • Analyzes structural elements
  • Generates PRF analysis report

Output Files

File Type Description Content
{genome_id}_mutation_summary.csv Mutation Summary All mutations found in the genome with details
{genome_id}_row_hot_mutations.csv Hot and Row Mutations Spaced and adjacent nucleotide substitutions
{genome_id}.txt Individual Genome Files Detailed mutation information for each genome
potential_PRF.csv Frameshift Analysis Potential frameshifting sites (if enabled)

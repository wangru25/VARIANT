Overview

VARIANT is a comprehensive virus mutation analysis tool designed to track mutations at both genome and protein levels. It provides detailed analysis of viral sequences, identifying mutations, hotspots, and potential frameshifting sites.

Key Capabilities

  • Multi-sequence alignment analysis
  • Mutation detection and classification
  • Hot and row mutation identification
  • Frameshift site detection
  • Protein-level impact analysis

Supported Viruses

  • Single-segment viruses (SARS-CoV-2, HIV-1, etc.)
  • Multi-segment viruses (Influenza H3N2, etc.)
  • Custom virus definitions

Features

Genome Analysis

Comprehensive mutation tracking across entire viral genomes

Protein Impact

Analysis of mutations on protein structure and function

Frameshift Detection

Identification of potential ribosomal frameshifting sites

Interactive Visualizations

Generate all three visualization types with one click

Streamlined Workflow
  • Auto-populated virus selection
  • One-click visualization generation

Advanced Visualizations

Quick Start

🚀 Get Started in 4 Steps

Follow these simple steps to analyze your virus data

Step 1: Data Upload

  1. 1Go to Data Upload tab
  2. 2Select virus type (or create custom)
  3. 3Upload reference genome (.fasta)
  4. 4Upload reference proteome (.fasta)
  5. 5Upload MSA file (.txt)

Step 2: Analysis & Visualization

  1. 1Go to Analysis & Visualization tab
  2. 2Select your virus (auto-populated)
  3. 3Optional: Enter specific genome ID
  4. 4Click "Start Analysis"

Step 3: Analysis & Visualizations

  1. 1Wait for analysis completion
  2. 2Click "Generate All Visualizations"
  3. 3View all three visualization types
  4. 4Use fullscreen for detailed view

Step 4: Track with Job ID

  1. 1Note your unique job ID
  2. 2Go to History tab
  3. 3Track job status and progress
  4. 4Download results when complete

Usage Guide

Analysis Options

Process All Genomes (Default)

Leave the Genome ID field empty to analyze all sequences in the MSA file. This is the recommended option for comprehensive analysis.

  • Generates files for each genome
  • Identifies common mutations
  • Provides complete analysis
Specific Genome ID

Enter a specific genome ID (e.g., EPI_ISL_123456) to analyze only that sequence.

  • Faster processing
  • Targeted analysis
  • Single genome focus

Advanced Features

Automatic Frameshift Detection

Automatically detects potential programmed ribosomal frameshifting sites in every analysis. This feature:

  • Identifies slippery sequences
  • Analyzes structural elements
  • Generates PRF analysis report
  • No additional configuration needed
Job Tracking System

Track your analysis progress with our job management system:

  • Real-time status updates
  • Unique job IDs for tracking
  • Download results when ready
  • History of all analyses
